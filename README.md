# DayTourity
Aplikasi platform digital untuk menghubungkan wisatawan dengan tour guide lokal terverifikasi secara praktis.

## Deskripsi Proyek
Aplikasi ini berfokus pada penyediaan layanan pemandu wisata yang terpercaya guna menggerakkan ekonomi warga lokal sekaligus memberikan pengalaman liburan yang berkualitas bagi turis. 

**Kontribusi terhadap SDG 8 (Decent Work and Economic Growth):**
* **Pekerjaan Layak:** Memberikan standar pendapatan dan pengakuan profesional bagi pemandu lokal melalui sistem verifikasi.
* **Pertumbuhan Ekonomi:** Digitalisasi sektor pariwisata mikro untuk memperluas akses pasar bagi warga setempat.

## Fitur Utama
Sesuai dengan tujuan proyek, fitur-fitur utama meliputi:
1. **Seamless Booking:** Sistem pemesanan jadwal tur langsung melalui satu platform digital.
2. **Advanced Search:** Pencarian guide berdasarkan lokasi, spesialisasi, dan bahasa.
3. **Cross-Platform Interface:** Antarmuka responsif yang dibangun dengan Flutter untuk Android dan iOS.
4. **Detailed Price-view:** Memaparkan harga secara detail untuk setiap aktivitas pada itinerary. 

## Technology Stack
* **Frontend:** Flutter (Dart)
* **Backend:** Nest.js (TypeScript)
* **Database:** PostgreSQL
